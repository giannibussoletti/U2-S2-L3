SASS

SASS, acronimo di Syntactically Awesome Style Sheet, è un pre-processore per CSS. SASS estende le possibilità di CSS facilitando e velocizzando lo sviluppo dei fogli di stile. Per pre-processore si intende una tecnologia che permette di ottenere fogli di stile ottimizzati, potenti, facilmente gestibili e mantenibili. Un pre-processore permette di estendere ai CSS le strutture di programmazione come variabili, funzioni, cicli e così via.

è un sistema per mantenere insomma il css in maniera più ordinata soprattutto quando si parla di grandi progetti e/o progetti a lungo termine
